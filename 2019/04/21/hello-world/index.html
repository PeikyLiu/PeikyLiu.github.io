如何搭建技术博客

hexo

1.安装hexo

npm install -g hexo-cli（全局安装）

2.cd到工作目录 (blog是你自己的文件夹名字)

hexo init blog

3.打开_config.yml 全局配置文件 修改theme属性

theme：next

4.本地跑起hexo

$ hexo g 打包项目（没上线前可不执行）
$ hexo s 启动本地web服务器

5.打开 http://localhost：4000/ 检查是否成功

推送到Github

6.修改hexo项目的根目录下的全局配置文件_config.yml

1 deploy:
2 type: git
3 repo: https://github.com/PeikyLiu/PeikyLiu.github.io
4 branch: master

7.为一键部署做准备 安装git hexo 发布插件

$ npm install hexo-deployer-git --save（显示问题前面是两个-）

8.上线

$ hexo clean 清理缓存
$ hexo g 打包项目
$ hexo d 一键部署

9.发布文章

采用md语法编写文章 原因简单易上手操作
由于hexo打包（hexo g）时候 会编译成 对应的html css js等线上部署的文章
那么 我们要对文章 和 项目的主体 进行相关的配置
# vue
